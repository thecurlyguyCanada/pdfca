# PDF Files Directory

This folder contains PDF files hosted on pdfcanada.ca.

## Upload Instructions

Upload your PDF files to this folder. They will be accessible at:
`https://www.pdfcanada.ca/pdfs/[filename].pdf`

## Current Files

- `surah-baqarah.pdf` - Surah Al-Baqarah (The Cow) - Full Quran Chapter 2

## Notes

- Keep file sizes reasonable (< 50MB recommended)
- Use lowercase filenames with hyphens
- PDFs are served via Vercel's CDN for fast delivery
